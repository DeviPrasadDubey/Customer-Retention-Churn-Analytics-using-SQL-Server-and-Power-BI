# Customer Retention & Churn Analytics

This project simulates a real-world business problem where the goal is to understand customer buying behavior, identify churned customers, and build data-driven insights to support retention strategies.

## Technologies Used
- SQL Server for data modeling and analytics
- Power BI for interactive dashboards

## Key KPIs
- Churn rate
- RFM segments
- Retention metrics

## Outcomes
- Identified customers likely to churn
- Segmented customer base for targeted campaigns
- Built dashboards for executive reporting.
